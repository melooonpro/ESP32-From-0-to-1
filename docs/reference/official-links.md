# 官方资料

官方资料像说明书，不像入门故事书。初学时不用从第一页读到最后一页；当你在某章跑通一个现象后，再来这里查对应 API，会更容易看懂。

本教程优先参考乐鑫公开官方资料和官方开源仓库。链接会随着教程迭代继续补充。
